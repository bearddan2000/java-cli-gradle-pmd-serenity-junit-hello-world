# java-cli-gradle-pmd-serenity-junit-hello-world

## Description
A POC for gradle app using junit5,
pmd and serenity plugins.

## Tech stack
- java
- gradle
  - junit
  - pmd
  - serenity

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
